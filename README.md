# SH Selección de Menús

App web para que los comensales de Compas Marine seleccionen sus menús mensuales.

## URLs

- **Comensal**: https://shgestion.github.io/seleccion-menus/
- **Admin**: https://shgestion.github.io/seleccion-menus/admin.html

## Stack

- Frontend: HTML + JS vanilla (sin dependencias, sin build)
- Backend: Google Apps Script (Web App)
- Base de datos: Google Sheets
- Hosting: GitHub Pages

## Estructura

- `index.html` — Login y selección de menús para comensales
- `admin.html` — Panel de reportes (protegido por token)
